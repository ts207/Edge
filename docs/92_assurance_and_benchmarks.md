# Assurance and benchmarks

This document is the stable landing point for verification and benchmark references.

## Assurance categories

- smoke workflows for bounded end-to-end execution
- contract tests for schemas and surfaces
- architecture tests for import and boundary rules
- regression tests for previously observed operational defects

## Current benchmark posture

The repo contains benchmark and audit generators for ontology coverage, detector coverage, regime routing, and thesis overlap artifacts. Generated outputs should be treated as derived artifacts rather than hand-edited sources.
