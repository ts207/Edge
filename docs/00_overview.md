# Overview

Edge is a bounded research-to-runtime system for event-conditioned crypto thesis generation.

## Canonical lifecycle

**discover → validate → promote → deploy**

## Units of meaning

- **Anchor**: the event family or trigger surface.
- **Filter**: context restrictions applied to the anchor.
- **Thesis**: the promoted runtime object eligible for live deployment.

## High-level rule

Research artifacts are not live artifacts. Only promoted theses pass into deployment surfaces.
